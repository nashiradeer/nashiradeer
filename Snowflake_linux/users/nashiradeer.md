## Bash

- Install `on-my-bash`.
- Copy `~/.oh-my-bash/themes/agnoster/agnoster.theme.sh` to `~/.oh-my-bash/custom/themes/nashi/nashi.theme.sh`
- Comment `[[ -z ${AG_NO_CONTEXT+x} ]] && prompt_context` in `build_prompt`
- Enable it with `OSH_THEME="nashi"` in `~/.bashrc`
